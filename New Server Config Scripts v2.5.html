INSTALL IonCube

Step 1

Install IonCube

wget -qO- http://downloads3.ioncube.com/loader_downloads/ioncube_loaders_lin_x86-64.tar.gz | tar xvz -C /tmp; sudo cp /tmp/ioncube/ioncube_loader_lin_7.2.so /opt/bitnami/php/lib/php/extensions/ && sudo nano /opt/bitnami/php/etc/php.ini

Step 2

Paste this at the top of PHP.INI

zend_extension = /opt/bitnami/php/lib/php/extensions/ioncube_loader_lin_7.2.so

Check extensions

Set S3 Bucket CORS Configuration

add

<

to the beggining of every html tag below 


    ?xml version="1.0" encoding="UTF-8"?>
    CORSConfiguration xmlns="http://s3.amazonaws.com/doc/2006-03-01/">
                    CORSRule>
    AllowedOrigin>*/AllowedOrigin>
    AllowedMethod>GET/AllowedMethod>
    MaxAgeSeconds>3000/MaxAgeSeconds>
    AllowedHeader>Authorization/AllowedHeader>
    AllowedHeader>Content-Length/AllowedHeader>
/CORSRule >
/CORSConfiguration>
